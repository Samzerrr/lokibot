# Loki Bot

Un bot Discord développé en JavaScript.

## Installation

1. Clonez ce dépôt
```bash
git clone https://github.com/Samzerrr/lokibot.git
```

2. Installez les dépendances
```bash
npm install
```

3. Configurez votre fichier d'environnement

4. Lancez le bot
```bash
npm start
```
